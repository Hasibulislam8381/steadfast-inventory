**Installation**

1.**Clone the repository**
   ```bash
   git clone https://github.com/Hasibulislam8381/steadfast-inventory.git
   cd steadfast-inventory

2. composer install
   npm install && npm run dev

3. cp .env.example .env
   php artisan key:generate

4.php artisan migrate

5.php artisan serve

6.(Last Step):
   Visit /register to create an account.
   The dashboard will show sales & inventory summary.
